## Setup ##
_(Section WIP)_

快速部署
```shell
$ git clone https://github.com/Equim-chan/csu-ems-api.git
$ cd csu-ems-api
$ npm install
$ npm start
```
默认会部署在2333端口上，如要修改端口，请使用参数`-p [value]`

获取帮助
```shell
$ node app.js -h
```
实际部署时请自行调用pm2或forever等deamon
